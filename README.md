# Customer Retention & Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn behavior and retention patterns using Python, Pandas, and Power BI.

The objective is to identify factors influencing customer churn and provide actionable insights to improve customer retention.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Power BI

## Dataset
Telco Customer Churn Dataset

## Key Performance Indicators (KPIs)
- Total Customers: 5042
- Churned Customers: 1336
- Retained Customers: 3706
- Average Monthly Charges: 61.43

## Analysis Performed
- Customer Churn Analysis
- Contract Type vs Churn
- Payment Method vs Churn
- Gender vs Churn
- Customer Tenure Analysis
- Monthly Charges Analysis

## Key Insights
- Month-to-Month contracts have the highest churn rate.
- Electronic Check users show maximum churn.
- Gender has minimal impact on churn.
- Customers with lower tenure are more likely to churn.
- Higher monthly charges are associated with higher churn.
